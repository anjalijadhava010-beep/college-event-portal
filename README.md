# 🎉 College Event Portal

A responsive College Event Management Portal built using React JS.

This project helps students discover college events, view event details, register for events, and manage their registrations.


## 🚀 Features

- View college events
- Event details page
- Student registration form
- Form validation
- Branch and year selection
- Store registration using LocalStorage
- View my registrations
- Cancel registration
- Responsive navbar


## 🛠️ Technologies Used

- React JS
- JavaScript
- CSS3
- React Router DOM
- LocalStorage
- Vite
